About MBTiles

MBTiles is a specification for storing map tiles in a single SQLite database file. This format provides:

  • Single-file distribution
  • Efficient tile access
  • Metadata support
  • Zoom level support
  • Offline viewing capability

Path C Implementation

This is the production-ready Path C implementation using GDAL for high-performance map tile generation. Supports:

  • MBTiles format
  • PMTiles format
  • Cloud-Optimized GeoTIFF (COG)
  • Large survey optimization
  • Web mercator projection

Sonar Sniffer Ecosystem

Sonar Sniffer processes sonar data and generates visualizations. CESARops provides complementary drift modeling for Search and Rescue operations. Together, they provide a complete SAR data analysis platform.
